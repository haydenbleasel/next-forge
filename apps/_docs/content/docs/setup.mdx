## Prerequisites

### Homebrew

Homebrew is a package manager for macOS. It's used to install the other dependencies.

You can install it with the following command:

```sh title="Terminal"
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### NVM `Optional`

NVM is a version manager for Node.js. It's used to install and switch between different versions of Node.js.

You can install it with the following command:

```sh title="Terminal"
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
```

### pnpm `Optional`

pnpm is a fast, disk space efficient package manager. It's used to install the dependencies.

You can install it with the following command:

```sh title="Terminal"
brew install pnpm
```

### Neon CLI

Neon is a database platform. The CLI is used to interact with the database.

You can install it with the following command:

```sh title="Terminal"
brew install neonctl
```

### Stripe CLI

Stripe is a payment platform. The CLI is used to interact with the Stripe API.

You can install it with the following command:

```sh title="Terminal"
brew install stripe/stripe-cli/stripe
```

## Cloning

First, scaffold the app with:

```sh title="Terminal"
pnpm create next-app --example https://github.com/haydenbleasel/next-forge
```

## Login

Login to Stripe with:

```sh title="Terminal"
stripe login
```

Login to Neon with:

```sh title="Terminal"
neonctl auth
```

Finally, run the development server with:

```sh title="Terminal"
pnpm dev
```

Open the localhost URLs with the relevant ports listed above to see the app, e.g.

- [http://localhost:3000/](http://localhost:3000/) — The main app.
- [http://localhost:3001/](http://localhost:3001/) — The website.
